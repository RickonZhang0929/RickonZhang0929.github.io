    强化学习笔记

    Rickon Zhang 8/21/2017 机器学习

    # 强化学习笔记

    # 综述

    # 背景

    定位：和有监督学习，无监督学习相对
    特点：无需数据集，自行尝试学习
    应用：游戏
    分类： 有模型和无模型，区别在于智能体是否知道系统
    基于奖励 最优奖励总和
    基于策略 找到最优策略

    # 对象：

    代理 环境 动作 策略 状态 奖励 目标
    设置好环境，通过状态来指导代理，代理通过马尔科夫决策过程，
    做出正确动作，获得奖励，将动作执行先后顺序形成策略，达到目标结束
    自动进行决策，连续决策，最大化积累reward

    # 决策：

    策略学习 详细指示 输入状态
    Q-learning 让代理自行学习 输入状态和动作

    # 局限：

    数据训练量大——效率低性能差
    领域特殊性——有特定的更优方案
    奖励函数设计——主观，局部最优解

    # 难题：

    无法收敛

    # 应用：

    小老鼠吃奶酪
